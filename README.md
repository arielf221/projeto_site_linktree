# 🔗 Projeto Linktree

Este é um projeto de página estilo **Linktree**, desenvolvido durante um curso da **Rocketseat**, com o objetivo de praticar fundamentos de desenvolvimento web utilizando **HTML, CSS e JavaScript**.

## 📸 Preview

![Preview do projeto](./assets/assets/preview.png)

---

## 🚀 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

---

## 💡 Funcionalidades

- Página responsiva
- Links para redes sociais
- Alternância de tema (dark/light mode)
- Design moderno e minimalista
- Animações suaves

---

## 📂 Estrutura do projeto

projeto_site_linktree/

- index.html
- style.css
- script.js
- assets/assets

## 📚 Aprendizados

Durante o desenvolvimento, foram aplicados conceitos como:

- Estruturação semântica com HTML
- Estilização com CSS (flexbox, responsividade)
- Interatividade com JavaScript
- Organização de arquivos
